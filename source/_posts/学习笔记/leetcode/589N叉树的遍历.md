#### N叉树的遍历

1. 递归

```java
public List<Integer> preorder(Node root) {
    List<Integer> list = new ArrayList<>();
    if (root!=null) {
        dfs(list, root);
    }
    return list;
}
public static void dfs(List<Integer> list , Node root){
    list.add(root.val);
    for (Node child : root.children) {
        dfs(list,child);
    }
}
```
