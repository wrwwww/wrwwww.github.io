路由间参数的传递

区别:

  1. query 传参配置的是 path，而 params 传参配置的是name，在 params中配置 path 无效
  2. query 在路由配置不需要设置参数，而 params 必须设置
  3. query 传递的参数会显示在地址栏中
  4. params传参刷新会无效，但是 query 会保存传递过来的值，刷新不变 ;
